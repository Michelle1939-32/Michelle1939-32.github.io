CNC Design

Violin DesignOctober 1, 2020

The sound of music In order to start this project, I had to be trained in order to learn how to properly use the Axiom CNC machine. This machine uses The Axiom CNC is a machine that is used carve materials to create a design. This machine is controlled by a remote and it uses VCarve in order to make the correct design. In order to use the machine properly, you place the material on the bed. Then, use the double-sided tape to stick it down and drill down the corners.

This project was leaning towards the design portion. I did my project using tinkercad, and I designed a violin. I chose the violin because I used to play it and it’s one of my favorite instruments. The most challenging part was figuring out how to assemble the piece and make it look as realistic as possible. Towards the end, I couldn't get the “strings” to bend the way I wanted to. In the end, I can say that I'm proud of the way it turned out.
