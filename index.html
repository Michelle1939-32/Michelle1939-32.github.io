ABOUT ME

Date, 9/3/2020

Heyy!My name is Michelle Menard and I'm currently a sophomore at Wheaton College. I'm from Stamford, Connecticut, and grew up in the Fairfield County area for most of my life. My dream is to become a NeoNatal surgeon. Since I will be encountering a ton of technology in the future, I wanted to know how it all worked. For example 3D printing, it has become a big tool in the medical field. I have worked with 3D printers before but I need to sharpen my skills.

This isn’t my first time dealing with HTML,CSS, and JavaScript. Two summers ago I participated in a program called Girls Who Code. I was an 8-week immersion program. I learned Python,HTML,CSS, JavaScript,Arduino C and Scratch. Over time I’ve grown to love it, it does get frustrating at times but I want to learn more and become fully fluent in Python.

Project Management

Make a Website, September 03, 2020

HTML! The main point of this project was to create a website to put our work on.

In the beginning, you had to create a GitHub account. After that, you had to create a repository in order to store the website data. Next, you open Gitbash and list your computer files (ls), change the directory to your desktop, clone the repository , enter the GitHub file. Once all that is done you must add, push, commit and enter your URL in the browser to check if it says "Hello World" on the page. If it successfully says hello world, you have to choose the template you want to use. After you need to open up a text editor (because I'm on a PC I used Atom) to start typing out your code. When you finish selecting and committing to your template, you can make your edits and customize the website to make it your own. Throughout this whole process, I kept running into error codes. For example, when making my 13 slots for each project, the borders kept repeating, and layering on top of each other created a ton of headings to be misplaced and all bunched up together. At one point I had to delete the file and start all over because Brackets wouldn't let me save my word and commit. Other than that making the website wasn't bad. Enjoy the final product!!
